## # Attributes and Methods of GroupBy

``` py
movies = pd.read_csv('Lec-5/imdb-top-1000.csv')
movies
```

### (i) len()

-  Find the total number of GroupBy

``` py
len(movies.groupby('Genre'))        # Output : 14
```

### (ii) size()

- Find items in each group

``` py
movies.groupby('Genre').size()

# Output

# Action       172
```
